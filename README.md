# Media Scanner

Утилита под Linux (C++17) для фонового сканирования медиафайлов. Собирает статистику по расширениям и раздает актуальный JSON через встроенный HTTP-сервер.

## Требования
* ОС: Linux
* CMake 3.14+
* Компилятор с поддержкой C++17

## Сборка

Библиотеки (`nlohmann/json`, `cpp-httplib`) скачиваются CMake автоматически.

```bash
mkdir build && cd build
cmake ..
make
```

## Запуск

По умолчанию сканируется текущая папка с интервалом в 10 секунд.

```bash
# Запуск с пользовательскими параметрами
./media_scanner --path /home/user/Downloads --interval 30
```

**Доступные флаги:**
* `-p, --path` — директория для сканирования
* `-i, --interval` — интервал обновления данных (в секундах)
* `-h, --help` — вывод справки

## Получение результатов (API)

Программа поднимает локальный сервер на порту `1234`. Для получения найденных файлов выполните GET-запрос:

```bash
curl http://localhost:1234/media_files
```

Пример ответа:
```json
{
    "audio": ["track.mp3"],
    "images": ["photo.jpg", "screenshot.png"],
    "video": ["movie.mp4"]
}
```
